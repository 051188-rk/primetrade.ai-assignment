# Backend (Node.js + Express)

## Setup

1. `cd backend`
2. `cp .env.example .env` and fill values (MONGO_URI, JWT_SECRET)
3. `npm install`
4. `npm run dev` (requires nodemon) or `npm start`

## Notes
- API base: `/api`
- Auth routes: `/api/auth`
- Tasks routes: `/api/tasks`
- User profile route: `/api/user`
